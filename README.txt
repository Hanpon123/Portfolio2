ขั้นตอนการ Run File
1. แตกไฟล์ zip ให้ได้โฟลเดอร์  'Hanpon'
2. ติดตั้งเข้าโปรแกรม Visual Studio Code
3. เลือกปุ่ม Extensions ทางซ้ายมือ (ปุ่มรูปสี่เหลี่ยม 4 รูป)
4. พิมพ์ 'Live Server' ในช่องค้นหาและติดตั้ง Live Sever
5. ไปที่เเถบคำสั่ง เเล้วเลือก File > Open Folder > แล้ว Folder 'Hanpon'
6. ไปที่แถบ Explorer (รูปกระดาษ 2 แผ่น) ใน Visual Studio Code เลือกแถบ Hanpon
7. เลือกไฟล์ index.html เพื่อเปิดไฟล์
8. คลิ๊กที่ปุ่ม Go Live ที่แถบสีฟ้าด้านล่างขวาของโปรแกรม เพื่อเปิดหน้าเว็บ

*หากมีปัญหาไม่สามารถเปิดไฟล์ได้สามารถเข้าดูเว็บได้ที่ ลิงค์ : https://hanpon.netlify.app/
